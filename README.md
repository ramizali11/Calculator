# Calculator
I Created a Basic Calculator With The Html And Python With The Help Of Libraries , I Used Two Library Flask And Simpleeval Which Help Me For Mathematical Operator. I Created It For Good Practice And To Understand The Module Flask 

# Feature
- Can Solve Basic Mathamatics Problem
- handle the error
- clean ui

# How To Run
1. Install Python
2. Install Flask
   pip install flask
3. Install Simpleeval
   pip install Simpleeval
5. Run the app
   python app.py
6. Open Link
   Open Local Host Link In Browser
